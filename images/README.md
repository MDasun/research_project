## Image Guidelines

- **Format**: JPG or PNG
- **Recommended Size**: 400x500px (portrait orientation)
- **File Size**: Keep under 500KB for optimal loading
- **Quality**: High resolution for clear profile photos
- **Background**: Preferably plain or professional background

## How to Add Images

1. Save your images with the exact filenames listed above
2. Place them in this `images/` folder
3. Images will automatically display on the About Us page

## Notes

- If an image file is missing, a placeholder will appear
- All images should be portrait orientation for best display
- Ensure all file names are lowercase with underscores (no spaces)
