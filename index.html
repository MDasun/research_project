Wearable IoT Research Project

Real-Time Soldier Health and Location Monitoring System

Battle Buddy 2.0 is a wearable frontline support system designed to monitor soldier vital signs, detect falls, track location, and transmit alerts to headquarters using secure long-range communication.

Explore Project Team Details

Core Features

  • Heart Rate, SpO₂, and Temperature Monitoring
  • Fall Detection using MPU6050
  • GPS Tracking with Remote Updates
  • LoRa-Based Long-Range Communication
  • Emergency SOS and Command Feedback

Domain

Literature Survey

Modern soldier support systems increasingly rely on IoT wearables, embedded sensors, and wireless telemetry to improve battlefield awareness. Prior studies show the value of continuous health monitoring, fall detection, and remote situational reporting, but many solutions are either short-range, power-intensive, or not designed for harsh field conditions.

Research Gap

Existing systems often lack an integrated wearable design that combines health monitoring, fall detection, location awareness, and long-range secure transmission in a practical field-ready format.

In addition, many existing monitoring solutions do not provide a lightweight and battlefield-ready visualization layer that combines health data and geolocation in one interface with very low latency.

Research Problem

Soldiers in remote or high-risk environments may suffer injuries, collapse, or lose communication without immediate support. Delayed identification of such incidents can reduce survival chances and compromise mission effectiveness.

A major challenge is converting incoming field data into clear, actionable intelligence through a reliable dashboard that remains responsive under limited bandwidth and unstable connectivity.

Research Objectives

  • Monitor vital signs in near real time.
  • Detect fall events and emergency conditions quickly.
  • Transmit alerts and status updates over long distances.
  • Provide a simple wearable architecture suitable for field use.
  • Support reliable and secure communication with headquarters.
  • Visualize soldier health and GPS data through an integrated live dashboard.

Methodology

The project uses a two-unit wearable architecture. The wrist unit captures health data using MAX30102 and MLX90614 sensors and sends processed readings via UART. The shoulder unit performs fall detection using MPU6050, acquires GPS coordinates, and forwards integrated data through SX1278 LoRa communication to the monitoring side.

For the visualization subsystem, the master ESP32 node forwards parsed data through Wi-Fi using MQTT or HTTP, Node-RED handles backend processing and streaming, and a dashboard presents live metrics, map-based tracking, and alert states.

Technologies Used

  • ESP32 Microcontrollers
  • MAX30102 Pulse Oximeter Sensor
  • MLX90614 Temperature Sensor
  • MPU6050 IMU Sensor
  • NEO-M8N GPS Module
  • SX1278 LoRa Module
  • Node-RED Backend Flows
  • MQTT / HTTP Protocols

Samarasinghe S C D – Individual Contribution

This project component focuses on the design and implementation of a secure long-range communication system using LoRa technology with multi-hop relay capability. It ensures reliable and protected data transmission between soldier units and the headquarters (HQ) in infrastructure-less and hostile environments.

  • Developed the LoRa communication framework using the SX1278 LoRa Module integrated with ESP32 (shoulder unit).
  • Designed and implemented a secure packet structure incorporating AES-256 encryption and HMAC-SHA256 authentication for confidentiality and integrity.
  • Implemented multi-hop relay communication to extend transmission range and overcome environmental obstacles using controlled hop count (HOP_MAX).
  • Developed communication flow from wrist unit → shoulder ESP32 → LoRa transmission → relay node → HQ system.
  • Integrated anti-replay protection mechanisms using packet counters and validation logic.
  • Optimized LoRa parameters (spreading factor, bandwidth, coding rate) to balance range, power consumption, and data reliability.

The novelty of this contribution lies in combining secure encryption mechanisms with multi-hop LoRa communication, enabling resilient, long-distance, and tamper-resistant data transmission suitable for battlefield environments.

Mudalige D H T H – Individual Contribution

This project component focuses on real-time soldier tracking using GPS integrated with predictive modeling through Kalman filtering and IMU sensor fusion. It ensures continuous and accurate location tracking even in scenarios where GPS signals are weak or unavailable.

  • Developed the GPS tracking system using the NEO-M8N GPS Module integrated with the ESP32 platform.
  • Implemented a Kalman filter-based predictive model to estimate position during GPS signal loss.
  • Integrated data from the MPU6050 Sensor to enhance motion tracking through acceleration and gyroscope measurements.
  • Designed algorithms for sensor fusion, combining GPS coordinates, velocity, and IMU-derived motion data.
  • Handled GPS outages using timeout mechanisms and predictive updates based on velocity and direction.
  • Converted GPS coordinates into Cartesian space for efficient Kalman filtering and motion estimation.

The novelty of this contribution lies in providing continuous and reliable soldier tracking through intelligent prediction, ensuring operational awareness even in GPS-denied or obstructed environments.

Kishoban R – Individual Contribution

This project component focuses on real-time health monitoring and fall detection using wearable IoT sensors, enabling automatic detection of critical conditions and emergency situations.

  • Developed the wearable sensor system using ESP32 integrated with the MAX30102 Sensor for heart rate and SpO₂ monitoring.
  • Integrated the MLX90614 Sensor for non-contact body temperature measurement.
  • Implemented fall detection using motion data from the MPU6050 Sensor based on acceleration thresholds and impact analysis.
  • Designed a multi-condition decision algorithm to identify abnormal health parameters and fall events.
  • Enabled automatic triggering of emergency alerts without requiring manual intervention.
  • Displayed real-time health data locally using an OLED interface and transmitted data to the communication module.

The novelty of this contribution lies in combining multi-parameter health monitoring with intelligent fall detection, enabling rapid emergency response and improving soldier survivability in high-risk environments.

Kumbukkotuwa K R D M - Individual Contribution

This project component focuses on real-time data visualization and the system interface for soldier monitoring. It receives parsed sensor data from the master ESP32 node and transforms it into an interactive dashboard for commanders and monitoring personnel.

  • Developed the real-time dashboard interface using React.js and Node-RED.
  • Displayed soldier health metrics such as heart rate, temperature, SpO₂, and GPS location.
  • Integrated MQTT/HTTP-based communication for low-latency data updates from the master ESP32 node.
  • Implemented intuitive gauges, map-based soldier tracking, and color-coded health indicators.
  • Optimized the dashboard for bandwidth-constrained environments using compact JSON payloads and asynchronous updates.

The novelty of this contribution is a lightweight yet responsive monitoring dashboard that supports real-time decision-making through map-based GPS tracking, health gauges, historical movement visibility, and traffic-light style status indicators.
